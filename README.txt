# pyファイルをまとめてテキスト化(GPT確認用) ReadMe

## 概要
「pyファイルをまとめてテキスト化(GPT確認用)」は、カレントディレクトリにあるPythonスクリプト（.pyファイル）を読み込み、その内容を1つのテキストファイルにまとめるユーティリティです。このツールは、特にプログラムの内容を一括して確認したい場合や、ドキュメント化の作業を行いたい場合に便利です。

## 使用方法
プログラムは実行可能な`.exe`形式で提供されており、以下の手順に従って使用します。

1. `pyファイルをまとめてテキスト化(GPT確認用).exe`をPythonファイルが存在するディレクトリに配置します。
2. コマンドプロンプトやバッチファイル、もしくはエクスプローラから直接`.exe`ファイルを実行します。
3. プログラムはカレントディレクトリ内の全ての`.py`ファイルを読み込みます。
4. 各ファイルの内容は「```ファイル名```」で区切られ、一枚のテキストファイルに連結されます。
5. 連結された内容は`GPT確認用Pythonファイル内容全文.txt`という名前のファイルに保存されます。

## 注意点
- 実行時にカレントディレクトリにのみ影響を与えるため、特定のフォルダ内の`.py`ファイルのみを扱います。
- Unicodeを含むファイルでも対応可能ですが、テキストエンコーディングに関する問題が発生する場合がありますのでご注意ください。
- セキュリティソフトウェアによっては、未知の`.exe`ファイルを実行することに危険性を指摘される可能性があります。信頼性を確認した上で実行を行ってください。
